# git commit

Do a commit to both mmtk-core and mmtk-openjdk repo, if there are any uncommitted changes (both staged or unstaged).

Add and commit the changes with:
* a short git commit message title and
* a relatively more comprehensive message body highlighting the changes.
* Do not mention other things, e.g. the details of your test process

NOTE:
* Commit only, do not push.
* You need to review the changes to understand what's changed.
* Don't commit other repos. mmtk-core and mmtk-openjdk only.
